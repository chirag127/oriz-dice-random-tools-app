# Oriz Dice — Random tools

> Browser-based random and picker tools under the **Dice** product brand of the Oriz family.

**Live at**: <https://random.oriz.in> · **Status**: scaffold

## What this is

Dice covers everyday randomness — dice rolls, coin flips, number generators, list shufflers, name pickers, and team splitters. All randomness is generated client-side; nothing is logged.
