# Oriz Dice — Random + Picker Tools

Browser-based random + picker tools under the **Dice** product brand (randomness) of the Oriz family.

**Live:** [https://random.oriz.in](https://random.oriz.in)

## Stack

- [Astro 6](https://astro.build/) — content + islands
- [React 19](https://react.dev/) — interactive components
- [Tailwind CSS v4](https://tailwindcss.com/) — styling
